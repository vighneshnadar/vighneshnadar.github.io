How to connect to AWS EC2 Windows Server 2019 Instance?

12 December, 2021 | Vighnesh | AWS

I hope you have created your first instance successfully.Now lets connect to our Instance,in this tutorial I will be using Windows Server Base 2019 as my Instance you can follow this tutorial if you are using any other version of Windows Server as well.

 

Now when your Instance is running successfully,Select the Instance and click Connect.Here choose RDP(Remote Desktop) Client and then click on Get Password,here you will need the Key-Value pair(Example.pem) file which you downloaded while creating Instance.

Once you have downloaded the RDP client and decrypted password you can connect to that RDP client which has your Windows Server using the decrypted password.

Open the RDP client which you downloaded and copy the decrypted password.

Click on connect and paste the password here.

 

Here we go we have succesfully connected to our Instance Part2 Example.
